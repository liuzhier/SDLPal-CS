# SDLPal-CS
# 使用 C# .NET 10 实现的经典中国游戏 PAL 的 mod 开发和调试工具
## 功能概述：
1.Mod 解打包工具使用 C# .NET 10 + Avalonia 开发

2.Rng、YJ_1/WinPack 解码模块来自 [sdlpal](https://github.com/sdlpal/sdlpal)

3.真正意义上的实现了所有数据全部可直接编辑，且数据结构明了（全部导出为 json 数据）

4.脚本部分使用了阉割语法后的 typescript 语言，面向有些许编程基础的群体

5.将所有图像导出为了 png-32，同时将 PAL.MKF 导出为 Photoshop 可用的 ACT 颜色表

6.地图编辑器是 [palresearch/MapEditor](https://github.com/palxex/palresearch/tree/master/MapEditor) Hack 过的版本

---
## 目前支持平台：
Windows，未来将支持 Mac、Linux、Android、IOS 等主流平台
